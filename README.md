# Portfolio

This is my web portfolio, developed with the following technologies. The aim is to show my technical and interpersonal skills in conjunction with my new learning of web technologies.

- HTML
- CSS
- TypeScript (ES6)
- Angular 19

To run this projetct, you need to:

1. Install Node Js: <https://nodejs.org>
2. Run the following command: `npm install -g @angular/cli` This command will be installing Angular packages in your computer
3. To run this application, in your root folder, run this command: `npm start run`
4. Have Fun
